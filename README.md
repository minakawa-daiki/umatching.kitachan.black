## モチベーション
「うまっちんぐ！」はウマ娘のルームマッチ機能をより快適に、よりウマ娘を楽しくプレイできるようにしたいというモチベーションからリリースしました。  
特にこれまでの他のサービスなどの大会機能は、視聴者は参加できないのが大多数でしたが、「うまっちんぐ！」では視聴者も投票（ウマチケ）ができることで全員参加型を目指しています。  
みなさんに最高にウマ娘を楽しむ環境を用意できればと思います。
